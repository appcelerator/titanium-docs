## Examples

### Create an ACL

This example creates an ACL object and checks the response.

    Cloud.ACLs.create({
        name: 'testACL',
        reader_ids: readers.ids.join(','),
        writer_ids: writers.ids.join(','),
        public_read: "false",
        public_write: "false"
    }, function (e) {
        if (e.success) {
            alert('Created!');
        } else {
            alert('Error:\n' +
                ((e.error && e.message) || JSON.stringify(e)));
        }
    });
